Various collections to store and retrieve your data.

## Requirements

PHP >= 7.2

## Installation

### Using composer
```bash
$ composer require sauls/collections
```
### Apppend the composer.json file manually
```json
{
    "require": {
        "sauls/collections": "^1.0"
    }
}
```
